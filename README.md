# Kafene
A comprehensive web-based administration panel designed for managing e-commerce or retail operations. This project features a multi-page interface that allows administrators to track real-time orders, manage inventory and view user profiles. It emphasizes a clean, data-driven UI with efficient filtering and search functionalities.
